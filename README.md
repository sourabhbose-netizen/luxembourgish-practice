# 🇱🇺 Lëtzebuergesch Practice

Sproochentest A1 preparation app — listen to questions, speak or type your answers, get instant feedback.

**Live app:** https://sourabhbose-netizen.github.io/luxembourgish-practice/

## Features
- 40+ questions across 10 categories (Greetings, Family, Work, Past Tense, etc.)
- Text-to-speech for hearing questions
- Speech recognition for spoken answers (Chrome, uses German as proxy)
- Type-to-answer for reliable spelling practice
- Fuzzy matching with phonetic similarity
- Score tracking and streaks
- Hints and answer reveal

## How to use
Open in Chrome for best experience. Click 🔊 to hear the question, then type or speak your answer. Vill Spaass! 🇱🇺
